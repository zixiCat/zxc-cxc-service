# ZXC CXC Service

## Description
- OpenAPI Spec(API First)
- JOOQ(Database First)

## Prerequisites
- Java 17
- Maven 3.X
- MySQL 8.X

## Init Database

Running MySQL8 Service,and initialize `zixiDatabase` database using the `init.sql` script
```sh
docker-compose up
```

Running the Application
```sh
mvn clean compile
```
